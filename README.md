# 🇮🇳 Bharat-Mitra

Bharat-Mitra is an AI-powered citizen assistance platform that helps users access government services, discover schemes, register complaints, and get instant assistance using Google's Gemini AI.

---

## 🚀 Features

- 🤖 AI Assistant (Google Gemini)
- 👤 User Registration & Login
- 📋 Complaint Registration
- 📜 Complaint History
- 🏛️ Government Scheme Finder
- 📊 Dashboard
- 📱 Responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router
- Axios

### Backend
- Node.js
- Express.js

### Database
- MySQL

### AI
- Google Gemini API

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/aman12-droid/Bharat-Mitra.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
